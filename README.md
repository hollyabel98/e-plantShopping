# e-plantShopping

## Paradise Nursery Shopping Application

This is a React-based web application for an online plant shop.  
Users can browse plants sorted into categories, add items to a shopping cart, and manage their selections.

### Features
- Display of plant categories with images and prices
- Add to Cart functionality with quantity updates
- Responsive design with a landing page and an 'About Us' section

### Technologies Used
- React
- Redux Toolkit
- Vite (for development server)

### How to Run
1. Clone the repository
2. Run `npm install` to install dependencies
3. Run `npm run preview` to start the app locally
